Domestic pharmaceutical flow spatial pattern

We applied the same analysis from Food flows between counties in the United States on the pharmaceutical product flow network. Following this paper, we computed centrality value of a node as the between-ness centrality value of the node, and connectivity value of a node as its node degree. According to this paper, a 'core' group of counties is evident which have both high connectivity and centrality and are critical to the structure of the flow network.
